# Cortex Tools

Optional utilities and development tools for Cortex.

## Available Tools

### `optimize_apple_silicon.sh`
Quick verification tool to check if Apple Silicon GPU acceleration is working properly. This runs the validation tests and provides optimization tips.

```bash
./tools/optimize_apple_silicon.sh
```

## Note

These tools are optional and not required for normal Cortex operation. The main `install.sh` script handles all necessary setup automatically.
